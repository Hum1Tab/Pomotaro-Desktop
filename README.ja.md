# Pomotaro 🍑 [English](./README.md)

Pomotaro（ポモタロウ）は、モダンで高機能なポモドーロ・タイマーアプリです。
単なるタイマーではなく、「1秒の狂いもない正確さ」と「中断しても再開できる安心感」を備え、あなたの集中（鬼退治）を強力にサポートする最高の「お供」になります。

## ✨ 強力な「お供」たち（主な機能）

アプリの中に、あなたの作業を支える3匹の強力なお供がいます：

- **🐕 忠実な「イヌ」：高精度タイマー**
  - 内部ロジックを終了時刻ベースに刷新。OSの負荷に左右されず、1秒の狂いもなく正確に時を刻みます。
  - アプリを閉じたりPCを再起動しても、**中断した箇所から自動で復旧**します。
- **🐒 賢い「サル」：Discord 連携**
  - 今何ポモドーロ目か、休憩中か集中中かを Discord のステータスにリアルタイムで表示。
  - 仲間と一緒に集中している感覚を味わえます。
- **🕊️ 鋭い「キジ」：タイトルバー通知**
  - アプリが背面に隠れていても、ウィンドウタイトルに残り時間を表示。
  - 作業の手を止めることなく、進捗をチラ見できます。

## 🛡️ データの守り（二重保存）

あなたの努力の証（学習記録）を消しません。
- **ハイブリッド保存**: ブラウザのキャッシュ(`localStorage`)だけでなく、PC内に物理ファイル(`app-data.json`)として自動保存。
- **自動復旧**: 万が一データが消えても、ファイルから自動で読み込んで元通りに復元します。
- **全部入りエクスポート**: 設定・タスク・履歴のすべてをまるごとバックアップ可能。

## 🎨 その他の特徴

- **心地よい音色**: ラグのないホワイトノイズ（White/Pink/Brown）と通知音。
- **スマートな操作**: インストール後すぐに使えるオートフォーカス、矢印キーによるセッション切替。
- **モダンな UI**: ダークモード、スムーズなアニメーション、集中を妨げない美しいデザイン。

## 🚀 使い方

### 開発者向けセットアップ

1. **依存関係のインストール**:
   ```bash
   pnpm install
   ```

2. **起動（開発モード）**:
   ```bash
   pnpm dev
   ```

3. **ビルド（本番用）**:
   ```bash
   pnpm build
   ```
